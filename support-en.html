Health Companion is a local-first personal health tracking and review tool.

Sleep did not import from Apple Health

Confirm that the selected date has sleep records and a data source in the system Health app, then run Sleep Import Diagnostics in this app's Settings. iOS does not tell apps whether an individual read permission was denied.

A reminder made no sound

Check notification permission, Sounds, Focus, Silent Mode, and Scheduled Summary. Health reminders are local notifications and are not the same as system alarms.

Backup and restore

Create a complete JSON backup in Settings and save it to Files. Choose that JSON file when restoring. CSV works well for viewing or migrating records, while JSON preserves settings and Wellness favorites.
